# CognitiveServices
Rama: 2-TextAnalytics

Esta rama está basada en el inicio rápido para utilizar Text Analytics mediante la biblioteca de cliente en C#. Las instrucciones completas las puedes encontrar en https://docs.microsoft.com/es-mx/azure/cognitive-services/text-analytics/quickstarts/client-libraries-rest-api?tabs=version-3-1&pivots=programming-language-csharp

Considera que como requisitos es necesario contar con una suscripción de Azure para crear un recurso de Text Analytics
