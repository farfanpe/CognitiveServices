# CognitiveServices
Rama 8: OCR

Esta rama está basada en el inicio rápido para utilizar OCR (reconocimiento óptico de caracteres) utlizando C#. Las instrucciones completas las puedes encontrar en  https://docs.microsoft.com/es-mx/azure/cognitive-services/computer-vision/quickstarts-sdk/client-library?tabs=visual-studio&pivots=programming-language-csharp

Considera que como requisitos es necesario contar con una suscripción de Azure para crear un recurso de Computer vision
